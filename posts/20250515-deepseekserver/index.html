    Introduction

    This post details a custom server build designed to run the Deepseek 71B model locally and serve as a high-capacity NAS for project documents. The goal is to provide a cost-effective, high-performance solution using consumer hardware available in Chile, while ensuring compatibility with Deepseek's demanding requirements and enterprise-grade storage needs. This guide is intended for IT professionals, researchers, and organizations interested in deploying advanced AI models and centralized storage on-premises.

    Solution Overview

    After analyzing Deepseek's hardware requirements (see previous post), I determined that running the 71B model locally requires a system with extremely high GPU VRAM, substantial system RAM, and fast storage. In addition, the server must function as a NAS, allowing secure access to project files from anywhere. The proposed solution leverages high-end consumer components for the best price/performance ratio, but requires careful selection and assembly due to compatibility and supply constraints in the Chilean market.
